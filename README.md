# homebrew-tengo-lsp

Homebrew tap for [tengo-lsp](https://github.com/popoffvg/tengo-lsp), a language
server for the [Tengo](https://github.com/d5/tengo) scripting language.

## Install

```sh
brew install popoffvg/tengo-lsp/tengo-lsp
```

Or tap first, then install:

```sh
brew tap popoffvg/tengo-lsp
brew install tengo-lsp
```

## Updating

```sh
brew update && brew upgrade tengo-lsp
```

> `Formula/tengo-lsp.rb` is generated automatically on each release by the
> `Release` workflow in `popoffvg/tengo-lsp`. Do not edit it by hand.
